# 📚 資料結構測驗系統 (Data Structure Quiz System)

這是一個基於 Django 開發的互動式測驗系統，專為學習資料結構（如：鏈結串列、堆疊與佇列）所設計。系統具備隨機抽題、錯題特訓、限時考試等多項完整功能。

## ✨ 系統核心功能
1. **🎲 動態隨機洗牌**：每次測驗動態打亂題目與選項順序，防止背誦答案位置。
2. **⏱️ 測驗雙模式**：支援「錯題補考」的一般練習模式，與「全域倒數」的正式考試模式。
3. **⌨️ 快捷鍵作答**：支援鍵盤 `1~5` 或 `A~E` 盲狙秒選，大幅提升測驗流暢度。
4. **💡 即時解析回饋**：作答後立刻顯示正確答案，並完美保留 Python 程式碼縮排進行概念解析。
5. **📊 歷史足跡追蹤**：完整記錄歷次測驗分數與耗時，並支援「單次作答明細」卡片式回顧。
6. **🎯 智能錯題本**：系統自動彙整答錯題目，針對弱點進行精準補考與針對性特訓。

## 🛠️ 環境建置與執行步驟 (Windows)

請依序執行以下步驟，即可在本機端順利啟動系統：

**1. 下載專案與開啟**
* 請至 GitHub 專案頁面點擊綠色 `Code` 按鈕，選擇 **`Download ZIP`** 下載專案。
* 將檔案解壓縮後，使用 **VS Code** 開啟該資料夾。
* 在 VS Code 中開啟終端機 (Terminal)。

（💡 提示：開啟終端機後，您可以直接一鍵複製下方所有指令，依序貼上執行即可）

```bash
# 2. 建立並啟動虛擬環境
python -m venv venv
.\venv\Scripts\activate

# 3. 安裝系統所需套件
pip install -r requirements.txt

# 4. 初始化資料庫結構
python manage.py migrate

# 5. 匯入預設題庫 (包含 11 題基礎題與 5 題原創題)
python import_data.py

# 6. 啟動本地伺服器
python manage.py runserver

#啟動後，請在瀏覽器輸入 http://127.0.0.1:8000/ 即可開始測驗！
